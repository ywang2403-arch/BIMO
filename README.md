# 🎮 BIMO
> Unity｜2D Pixel Platform Adventure


## 📖 项目简介
BIMO 是一款使用 Unity 开发的 2D 像素平台冒险游戏。玩家将扮演来自外星的角色 BIMO，在一次意外中身体部件散落至三个风格迥异的地图中。玩家需要穿梭于不同场景，通过探索、跳跃、收集及躲避敌人等方式寻找遗失的身体部件，最终帮助 BIMO 重组身体并返回自己的家园。
游戏以轻松治愈的像素美术风格为基础，结合平台跳跃、探索收集及环境叙事，引导玩家在不同地图中逐步推进剧情，并通过各具特色的关卡设计营造丰富的探索体验。

## 💻 我的职责
在三人团队中，我主要负责：
* 负责 City 地图的整体关卡设计、场景搭建及视觉风格设计。
* 使用 Unity（C#） 完成部分游戏交互逻辑及玩法功能开发。

* 参与前期调研、玩法设计、项目测试及版本迭代，根据测试反馈持续优化玩家体验。
* 在项目展示（Presentation）中负责讲解设计理念、关卡设计思路及优化过程，与团队共同完成项目汇报。
