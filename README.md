# 🎮 BIMO
> Unity｜2D Pixel Platform Adventure


## 📖 项目简介 （Project Overview)
BIMO 是一款使用 Unity 开发的 2D 像素平台冒险游戏。玩家将扮演来自外星的角色 BIMO，在一次意外中身体部件散落至三个风格迥异的地图（城市，天空，宇宙）中。玩家需要穿梭于不同场景，通过探索、跳跃、收集及躲避敌人等方式寻找遗失的身体部件，最终帮助 BIMO 重组身体并返回自己的家园。
游戏以轻松治愈的像素美术风格为基础，结合平台跳跃、探索收集及环境叙事，引导玩家在不同地图中逐步推进剧情，并通过各具特色的关卡设计营造丰富的探索体验。

---

## 🎯 项目目标 (Our Goal)
我们希望通过简单易上手的平台跳跃玩法与环境探索机制，为玩家营造轻松、治愈的冒险体验。项目在关卡设计中注重探索节奏与玩家引导，通过不同地图的视觉风格及互动设计提升游戏的新鲜感，并结合玩家测试反馈不断优化操作体验与关卡流程。

## 📓 设计迭代（Design Iteration）
City关卡设计：
初版地图以平台跳跃为主，但测试过程中发现玩家容易忽略收集目标，探索路线较为单一。因此重新调整了场景布局，在关键位置增加视觉引导，并优化了道具分布，使玩家能够更自然地完成探索流程。

敌人追逐机制：
初期NPC追逐速度过快，导致玩家频繁失败，影响游戏体验。通过降低移动速度、调整追击范围以及增加安全区域，使挑战性与可玩性达到更好的平衡。

玩家测试：
邀请同学进行试玩，根据反馈发现部分交互提示不够明显，因此补充了引导信息并优化了交互逻辑，提高了玩家对游戏目标的理解。
